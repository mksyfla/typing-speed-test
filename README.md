# Typing Speed Test

A typing speed test built with **Vanilla TypeScript**, **Vite**, and **Tailwind CSS**, inspired by the Frontend Mentor Typing Speed Test challenge.

https://www.frontendmentor.io/challenges/typing-speed-test

The application measures your typing performance by tracking metrics such as words per minute (WPM), accuracy, and other typing statistics while providing a clean, responsive user interface.

The goal of this project is to practice building interactive web applications using modern frontend tooling while keeping the implementation framework-free.


## Features
* Real-time typing speed calculation (WPM)
* Accuracy tracking
* Live typing statistics
* Responsive design for desktop and mobile
* Fast development and build process with Vite
* Utility-first styling with Tailwind CSS
* Built with vanilla TypeScript

## Tech Stack

* **TypeScript** – Application logic and type safety
* **Tailwind CSS** – Styling
* **Vite** – Development server and bundler

## Getting Started

### Prerequisites

Make sure you have the following installed:

* **Node.js** (v18 or later recommended)
* **npm** (comes with Node.js)

### Installation

Clone the repository:

```bash
git clone https://github.com/mksyla/typing-speed-test.git
```

Navigate to the project directory:

```bash
cd typing-speed-test
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit the local URL shown in the terminal (typically `http://localhost:5173`).

## How to Use

1. Select your preferred typing settings (difficulty, mode, or duration if available).
2. Click **Start** or begin typing when the test starts.
3. Type the displayed text as accurately and quickly as possible.
4. Your statistics, including **Words Per Minute (WPM)** and **Accuracy**, are updated in real time.
5. When the test finishes, review your results and try again to improve your personal best.
